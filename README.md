# Austin Crime Data Analysis – Power BI

## Project Overview

This project analyzes the Austin Crime dataset obtained from Data.gov using Power BI.  
The objective of this analysis is to identify patterns, trends, and relationships in reported crime incidents and generate meaningful, data-driven insights.

The focus of this project is analytical depth and interpretation rather than purely visual presentation. The findings aim to support informed decision-making related to public safety and resource allocation.

---

## Research Questions

1. What are the dominant crime types in Austin?
2. What trend do crime incidents show over time?
3. How do crime levels compare across council districts?
4. Is there a relationship between crime type and council district?
5. What unexpected or surprising patterns appear in the data?

---

## Key Insights

- A small number of crime types, including Family Disturbance, Burglary of Vehicle, and Theft, account for a significant portion of total incidents.
- Crime levels fluctuate seasonally, with higher incident counts during late spring and summer months.
- Crime is unevenly distributed across council districts.
- Certain crime types are concentrated in specific districts.
- Public spaces such as roadways, streets, and sidewalks account for a substantial share of reported incidents.
